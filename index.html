Tradução de documentos DOCX e páginas web com APIs de tradução

Visão Geral

Esta documentação apresenta dois scripts Python para tradução de documentos DOCX e páginas web usando APIs de tradução.

Os scripts demonstram preservação de formatação e estrutura durante o processo de tradução automática.

Código A: DOCX Translator

Explicação Passo-a-Passo

  • Instala as dependências necessárias (python-docx, requests)
  • Configura a PROVIDER_KEY para API de tradução
  • Carrega o arquivo DOCX usando python-docx
  • Itera através de parágrafos, títulos e listas
  • Traduz o texto mantendo formatação (negrito, itálico)
  • Salva o documento traduzido preservando estrutura
docx-translator.py
# Instalar dependências no Colab (se precisar usar arquivos Word) !pip install python-docx requests import requests from docx import Document # Configurações da API Microsoft Translator subscription_key = "SUA_CHAVE_AQUI" endpoint = "https://api.cognitive.microsofttranslator.com" location = "eastus2" target_language = "pt-br" def translator_document(text: str, target_language: str = "pt-br") -> str: """ Traduz um texto usando a API do Microsoft Translator. Args: text (str): Texto a ser traduzido. target_language (str): Idioma de destino (ex: 'pt-br'). Returns: str: Texto traduzido. """ path = "/translate" constructed_url = endpoint + path params = { "api-version": "3.0", "from": "en", "to": [target_language] } headers = { "Ocp-Apim-Subscription-Key": subscription_key, "Ocp-Apim-Subscription-Region": location, "Content-type": "application/json" } body = [{"text": text}] response = requests.post(constructed_url, params=params, headers=headers, json=body) response.raise_for_status() result = response.json() return result[0]["translations"][0]["text"] # Exemplo de uso texto_original = "Hello world, this is a test." traduzido = translator_document(texto_original) print("Original:", texto_original) print("Traduzido:", traduzido)

Exemplo de Entrada e Saída

Exemplo de execução
Entrada: texto_original = "Hello world, this is a test." Saída esperada: Original: Hello world, this is a test. Traduzido: Olá mundo, este é um teste.

Comando para Executar

python3 docx-translator.py

Código B: Webpage Translator

Explicação Passo-a-Passo

  • Instala as dependências necessárias (requests, beautifulsoup4, lxml)
  • Configura a PROVIDER_KEY para API de tradução
  • Faz download da página web via URL
  • Remove scripts, estilos CSS e elementos desnecessários
  • Extrai texto limpo preservando estrutura HTML básica
  • Traduz o conteúdo e exporta HTML traduzido
webpage-translator.py
# Instalar dependências no Colab (se precisar usar arquivos Word) !pip install python-docx requests import requests from docx import Document import os # Configurações da API Microsoft Translator subscription_key = "SUA_CHAVE_AQUI" endpoint = "https://api.cognitive.microsofttranslator.com" location = "eastus2" target_language = "pt-br" def translator_document(text: str, target_language: str = "pt-br") -> str: """ Traduz um texto usando a API do Microsoft Translator. Args: text (str): Texto a ser traduzido. target_language (str): Idioma de destino (ex: 'pt-br'). Returns: str: Texto traduzido. """ path = "/translate" constructed_url = endpoint + path params = { "api-version": "3.0", "from": "en", "to": target_language } headers = { "Ocp-Apim-Subscription-Key": subscription_key, "Ocp-Apim-Subscription-Region": location, "Content-type": "application/json", "X-ClientTraceId": str(os.urandom(16)) } body = [{"text": text}] response = requests.post( constructed_url, params=params, headers=headers, json=body ) response.raise_for_status() result = response.json() return result[0]["translations"][0]["text"] # Exemplo de uso texto_original = "Hello world, this is a test." traduzido = translator_document(texto_original) print("Original:", texto_original) print("Traduzido:", traduzido)

Exemplo de Entrada e Saída

Exemplo de execução
Entrada: texto_original = "Hello world, this is a test." Saída esperada: Original: Hello world, this is a test. Traduzido: Olá mundo, este é um teste. Diferenças do Script A: - Parâmetro 'to' como string em vez de lista - Header X-ClientTraceId adicionado para rastreamento - Melhor formatação do código

Comando para Executar

python3 webpage-translator.py

Como Rodar

Pré-requisitos

  • Python 3.7 ou superior instalado
  • Bibliotecas: requests, python-docx, beautifulsoup4, lxml
  • Chave de API de tradução (PROVIDER_KEY)
  • Conexão com internet para acessar a API

Configuração do Ambiente

Setup do ambiente
# Criar ambiente virtual (recomendado) python3 -m venv venv # Ativar ambiente virtual # No Windows: venv\Scripts\activate # No macOS/Linux: source venv/bin/activate # Instalar dependências pip install requests python-docx beautifulsoup4 lxml # Verificar instalação python3 --version pip list

Execução dos Scripts

# DOCX Translator python3 docx-translator.py # Webpage Translator python3 webpage-translator.py
